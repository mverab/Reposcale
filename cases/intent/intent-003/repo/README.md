# SuperSite CMS

A modern static site generator with plugin support and themes.

## Features
- **Plugin system**: Extend functionality with custom plugins
- **Theme engine**: Beautiful themes with template compilation
- **Plugin marketplace**: Browse and install community plugins
- **Asset pipeline**: Automatic CSS/JS bundling

## Quick Start
```bash
./scripts/build.sh
```

## Configuration
Create a `site.yaml` file (see v2 docs) or `site.json` (legacy).
